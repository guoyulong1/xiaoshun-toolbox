# 项目截图 | Project Screenshots

这个目录用于存放项目的截图文件。

## 📸 需要的截图文件

请添加以下截图文件到此目录：

1. **home.png** - 主页界面截图
   - 建议尺寸：1200x800 或更高
   - 展示工具箱的主页和导航

2. **time-tool.png** - 时间转换工具截图
   - 展示时间戳转换功能的使用界面
   - 包含输入和输出示例

3. **path-planning.png** - 路径规划工具截图
   - 展示路径规划的可视化界面
   - 包含网格、起点、终点和路径

4. **dark-mode.png** - 深色模式截图
   - 展示深色主题下的界面效果
   - 可以是任意工具页面的深色模式

## 📝 截图建议

- **分辨率**：建议使用 1200x800 或更高分辨率
- **格式**：PNG 格式，保证清晰度
- **内容**：展示实际使用场景，包含有意义的数据
- **浏览器**：建议使用 Chrome 或 Edge 浏览器截图
- **窗口大小**：确保界面完整显示，避免截断

## 🛠️ 如何获取截图

1. 启动开发服务器：`npm run dev`
2. 在浏览器中访问：`http://localhost:5174`
3. 导航到不同的工具页面
4. 使用浏览器开发者工具或截图工具获取高质量截图
5. 将截图文件重命名并保存到此目录

## 🎨 截图优化建议

- 确保界面干净整洁
- 填入有意义的示例数据
- 展示工具的核心功能
- 保持一致的浏览器窗口大小
- 考虑添加一些交互状态（如悬停效果）
