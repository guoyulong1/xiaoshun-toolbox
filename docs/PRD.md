# DevKit 工具箱 - 产品需求文档（PRD）

## 1. 背景与目标
为开发者与机器人工程师提供一个集成的网页工具箱，覆盖常见的数据格式转换、编码/解码、路径规划算法可视化以及机器人调试相关工具，提升日常工作效率。

## 2. 用户场景
- 后端/前端开发：需要快速进行时间戳与时间互转、进制转换、Base64/URL/Gzip 编解码。
- 机器人工程师：路径规划算法快速验证与可视化、CRC 校验与十六进制查看。
- 测试工程师：数据报文解析与校验。

## 3. 核心功能（MVP）
1) 时间转换
   - 时间戳（秒/毫秒） ↔ 时间字符串（本地/UTC）
   - 支持格式化（ISO、YYYY-MM-DD HH:mm:ss、本地自定义格式）
   - 复制结果与快捷填充当前时间

2) 进制转换
   - 2/8/10/16 互转
   - 支持大整数（BigInt）与负数
   - 字符串 ↔ 字节视图（Hex/ASCII）

3) 编码/解码
   - Base64 编码/解码
   - URL 编码/解码
   - Gzip/Zlib 压缩与解压（基于 pako）

4) 路径规划模拟
   - 网格地图编辑：设置障碍、起点、终点
   - 算法选择：BFS、Dijkstra、A*
   - 可视化展示：路径与搜索过程（热力图）

5) 机器人调试工具
   - CRC16/CRC32 计算器（常用多种多项式）
   - Hex 检查器：字节、ASCII、十进制视图

## 4. 非功能性需求
- 响应式 UI，简洁易用
- 本地运行，无需后端服务
- 良好的错误提示与输入校验

## 5. 里程碑
- M1：项目初始化与基础布局
- M2：时间、进制、编码工具完成
- M3：路径规划模拟与机器人工具初版
- M4：文档与测试、打包部署

## 6. 风险与对策
- 浏览器兼容性：采用现代浏览器 API，避免过时特性
- 大数据处理性能：在前端进行分块与异步处理（后续迭代）
